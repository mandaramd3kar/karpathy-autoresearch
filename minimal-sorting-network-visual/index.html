What is a sorting network?

A sorting network is a fixed sequence of comparisons — no branching, no decisions based on the data. The same wires, the same gates, every time. That's what makes them so useful for GPUs and hardware: they're parallel, predictable, and blazingly fast.

A single comparator: two wires, one gate
If left > right, swap. Otherwise, don't.

Why seven comparisons matter

⚡

GPU Kernels

Deep learning models sort small batches millions of times per second. Shaving comparisons from inner loops is measurable at scale.

🔧

Compilers

GCC and LLVM use sorting networks to generate optimal code for small fixed-size sorts — embedded in auto-vectorization passes.

🧬

Theory

Proving optimal sorting networks requires new lower bound techniques. Solutions here often transfer to other circuit complexity problems.
